# NEXT Theme -- Single
> 🎈 一个简洁大气，含夜间模式的 NEXT 博客主题
> 
> 🎈 A simple and modern theme for **NEXT Space** with night mode.

## Features

- [X] ⌨️ `Command` + `K` 打开工具箱 (Windows use `ctrl` + `K`, if bug, won't be fixed)
  - [X] Toggle Site Theme (light / dark)
  - [ ] Resently Updated
  - [ ] Posts Search
  - [ ] Friendly, Customly Keybindings
